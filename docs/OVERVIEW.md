# StormeAI Overview

StormeAI is a chat-only AI agent SaaS for organizations.

## Core promise

Patients get instant chat answers and appointment help. Organizations reduce repetitive front-desk work and capture more bookings 24/7.

## What it does

- Answers organization FAQs from organization-approved knowledge sources
- Explains services, hours, location, prices, and policies
- Collects appointment details through chat
- Creates/reschedules/cancels appointments
- Sends confirmation and reminder emails
- Routes appointment and handoff work inside the StormeAI dashboard
- Routes complex or sensitive cases to organization staff

## What it must not do

- Diagnose patients
- Prescribe medicine
- Replace doctors
- Handle emergency medical care
- Pretend to be a human agent

## Primary customers

- Dental organizations
- Dermatology organizations
- Aesthetic organizations
- Wellness organizations
- Therapy organizations
- Diagnostic/lab organizations
- Small private medical practices

## Differentiators

- Chat-only focus
- Organization-specific RAG
- Appointment-first UX
- Local Ollama default for development/private deployments
- Optional OpenAI/Claude provider routing
- Built-in appointment and handoff flows
- Organization-specific dashboard routing
- Fixed manual billing at ₱10,000/month with unlimited chats
