# StormeAI Modern Page Designs

This design pack contains Figma-importable SVG mockups for a more modern, user-friendly, configuration-heavy StormeAI SaaS experience.

## Design goal

StormeAI should feel like a premium organization operations cockpit: calm, safe, easy to configure, and focused on turning patient chats into appointments.

## Pages included

1. Landing Page
2. Auth / Sign In
3. Organization Onboarding
4. Dashboard Overview
5. AI Agent Command Center
6. AI Personality & Prompt Configuration
7. AI Provider Configuration
8. Knowledge Base & RAG Sources
9. Appointment Scheduling Configuration
10. Appointments Inbox
11. Patient Chat Widget States
12. Workflow Builder
13. n8n Integration Settings
14. Manual Billing & Plan Limits
15. Analytics & Insights
16. Safety, Escalation & Human Handoff

## How to use in Figma

1. Open the StormeAI Figma file.
2. Create pages/sections matching the list above.
3. Drag each SVG from `docs/design/figma-import/` into Figma.
4. Refine SVG groups into native Figma components as needed.

## Design principles

- Modern SaaS polish, not generic admin UI
- Friendly organization-specific language
- Clear configuration controls
- Safety and handoff always visible
- Appointment-first UX
- Tenant-aware organization settings
- Calm trusted blue + teal palette
- Rounded surfaces, soft shadows, clear hierarchy

## Configuration areas represented

- AI agent personality
- System prompt and safety rules
- AI provider routing: Ollama, OpenAI, Claude
- RAG source settings
- Appointment rules
- n8n webhooks
- Manual billing/plan limits
- Human escalation and emergency flow
