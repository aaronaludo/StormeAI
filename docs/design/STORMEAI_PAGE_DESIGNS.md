# StormeAI Page Designs

This design pack contains Figma-importable SVG mockups for the main StormeAI SaaS pages.

## Pages included

1. Landing Page
2. Auth / Sign In
3. Clinic Onboarding
4. Dashboard Overview
5. AI Receptionist Settings
6. Knowledge Base
7. Appointments
8. Patient Chat Widget
9. Workflow Builder
10. Billing / Manual Subscription
11. Analytics

## How to use in Figma

1. Open the StormeAI Figma file.
2. Create pages/sections matching the list above.
3. Drag each SVG from `docs/design/figma-import/` into Figma.
4. Convert/refine into components as needed.

## Design direction

- Calm clinic SaaS style
- Light background with blue/teal accents
- Rounded cards
- Clear status badges
- No overly medical/diagnosis visuals
- Chat-only receptionist positioning
